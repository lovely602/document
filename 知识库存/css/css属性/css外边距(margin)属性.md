### margin 属性
简写属性在一个声明中设置所有外边距属性。该属性可以有1到4个值

属性值
- auto 浏览器计算外边距
- length 规定以具体单位计的外边距值，比如像素、厘米等。默认值是 0px
- % 规定基于父元素的宽度的百分比的外边距
- inherit 规定应该从父元素继承外边距
````
margin:10px 5px 15px 20px;
margin:10px 5px 15px;
margin:10px 5px;
margin:10px;
````
### margin-bottom 
设置元素的下外边距
### margin-top 
设置元素的上外边距
### margin-left 
设置元素的左外边距
### margin-right 
设置元素的右外边距
