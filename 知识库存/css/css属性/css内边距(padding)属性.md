## padding 属性
简写属性在一个声明中设置所有填充属性。该属性可以有1到4个值

属性值
+ length 规定以具体单位计的填充值，比如像素、厘米等。默认值是 0px
+ % 规定基于父元素的宽度的百分比的填充
+ inherit 指定应该从父元素继承padding
````
padding:10px 5px 15px 20px;
padding:10px 5px 15px;
padding:10px 5px;
padding:10px;
````

### padding-bottom
设置元素的底填充

### padding-top
设置元素的顶填充

### padding-left
设置元素的左填充

### padding-right
设置元素的右填充
