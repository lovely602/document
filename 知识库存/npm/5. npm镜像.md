### 镜像源链接切换

- 全局切换镜像源
````
npm config set registry http://registry.npm.taobao.org
````
- 查看镜像源使用状态
````
npm get registry
或
npm config get registry
````
- 全局切换官方镜像源
````
npm config set registry http://www.npmjs.org
````

### 使用nrm切换镜像源
- 下载nrm
````
npm install -g nrm
````

- 查看可切换的镜像源： 
````
nrm ls (*表示正在使用的镜像源)
````

- 切换淘宝镜像源：
````
nrm use taobao
````
