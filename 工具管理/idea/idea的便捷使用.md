## 增大idea的运行内存
修改idea64.exe.vmoptions该文件中的Xmx的大小。
````
-Xms128m
-Xmx1024m
````
## idea快捷键的设置
1. 退回上一次编辑的地方
  Back  设置 Ctrl + CapsLock 快捷键
2. 代码提示
  Basic 设置 Alt + /
3. 重命名
  Rename 设置 F2
4. 快速修复
  Show Intention Actions   设置 Ctrl + 1
